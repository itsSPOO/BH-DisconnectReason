# DisconnectReason

**Author:** SPOO

## Overview
Customize/handle player disconnect reasons.

## Features
- Config-driven messages
- Lightweight resource

## Dependencies
- `qb-core`

## Installation
1. Copy the folder to your server `resources` directory.
2. Add to `server.cfg`:
   - `ensure DisconnectReason`

## Configuration
- `config.lua`

## Support
Open an issue / ticket with a short description and your console logs.
